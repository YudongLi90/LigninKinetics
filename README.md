# Lignin Kinetics code for low severity alkaline lignin extraction



[![DOI](https://zenodo.org/badge/1196619833.svg)](https://doi.org/10.5281/zenodo.19341240)


## How to use

Create envirionment by running:

`conda env create -f environment.yml`

Then activate environment:

`conda activate dak`

Now you can run the demo notebook `demo.ipynb` with `dak` environment
